Overview

Homework at Shevington has been shaped by feedback from parents/carers, students, and teachers.

We aim to use homework to:

  • Support student progression from KS3 to GCSE.
  • Encourage independent revision and exam preparation.
  • Counteract forgetting by revisiting and practising classroom knowledge.

Key benefits include improved learning, better exam preparation habits, and clear expectations. In accordance with guidance from the Education Endowment Foundation (EEF) we strive to ensure homework is effectively linked to classroom work and consistent.

Key Stage 3 Outline (Years 7-9)

What tasks might be set?

  • Tasks may include knowledge organiser retrieval, online platform tasks, quizzing, reading, comprehension, and writing projects.

How much time is expected to be spent on homework?

  • Weekly assignments for each subject area, with specific tasks and quizzes outlined per subject.
  • Core subjects (English, Maths & Science) will set up to 30 minutes of homework per week.  All other subject areas will set up to 15 minutes per week

Where is the homework set?

  • Each subject area has identified how homework is set via platforms like ClassCharts and monitored through tools like Google Classroom for quizzes and progress tracking.

Additionally:

  • There may be occasions when students are working on SAIL missions which involve completing work at home.

Key Stage 4 Outline (Years 10-11)

What tasks might be set?

Each subject will provide a half-term homework overview, covering:

  • Aims and objectives
  • Types of tasks
  • Assessment guidance
  • Homework access and submission details
  • Links to revision resources

Each subject are will also provide a Knowledge Organiser for each half term or topic.

How much time is expected to be spent on homework?

  • Weekly assignments for each GCSE subject.
  • Five hours weekly: one hour each for core subjects (English, Maths, Science) and thirty minutes for each GCSE option subject.
  • Extra time should be used for revision if tasks are completed early.

Where is the homework set?

  • Each subject area has identified how homework is set via platforms like ClassCharts and monitored through tools like Google Classroom for quizzes and progress tracking.

Additionally:

  • There may be occasions when students are working on SAIL missions which involve completing work at home.
